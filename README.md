# House Price Prediction

## Overview

This project aims to predict house sale prices using machine learning techniques. The workflow includes data preprocessing, exploratory data analysis (EDA), feature engineering, missing value treatment, model training, and performance evaluation.

## Dataset

* Train and test datasets containing housing features such as lot area, neighborhood, house quality, year built, garage details, and other property attributes.
* Target Variable: **SalePrice**

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-Learn
* XGBoost

## Project Workflow

1. Data Cleaning and Missing Value Handling
2. Exploratory Data Analysis (EDA)
3. Feature Engineering
4. Categorical Feature Encoding
5. Model Training and Evaluation
6. House Price Prediction

## Models Used

* Linear Regression
* Random Forest Regressor
* Gradient Boosting Regressor
* XGBoost Regressor

## Evaluation Metric

* RMSE (Root Mean Squared Error)

## Key Highlights

* Performed extensive feature engineering and preprocessing.
* Compared multiple regression models to identify the best-performing approach.
* Generated predictions for unseen housing data.
